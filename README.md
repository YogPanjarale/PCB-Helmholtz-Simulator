# PCB Helmholtz Coil Designer

A browser-based tool for exploring PCB Helmholtz coil designs and visualizing their magnetic field.

## Features

* Circular and square PCB coils
Field uniformity analysis
* Multi-layer and stacked PCB support
* Interactive 2D and 3D visualizations
* Real-time Biot–Savart field calculations
* IPC-2221 current estimation

## Usage

Open `pcb_helmholtz_designer.html` in any modern web browser and play with the settings to get the desired output
You can also try ideal distances/values
